# Rezepte

## Brot

Getestet mit Panasonic SD-ZB2502BXA (Brotbachautomat)

1 Tl = 1 Teelöffel

1 El = 1 Esslöffel


### Dinkelbrot
| Menge | Zutat |
| --- | --- |
| 3 dl | Wasser |
| 1.5 Tl | Salz |
| 1 Tl | Zucker |
| 300 g | Dinkel Mehl Hell (mit oder ohne Schrot) |
| 100 g | Dinkel Mehl Vollkorn |
| 7 g | Trocken Hefe |


### Kartoffelbrot
| Menge | Zutat |
| --- | --- |
| 6 dl | Wasser |
| 2 Tl | Salz |
| 1.5 Tl | Zucker |
| 500 g | Weissmehl oder Dinkel mit Schrot |
| 95 g | Kartoffelstock / Stocki |
| 7 g | Trocken Hefe |


### Pizzateig
| Menge | Zutat |
| --- | --- |
|  2.5 dl | Wasser |
| 2 Tl | Salz |
| 1.5 Tl | Zucker |
| 500 g | Mehl |
| 7 g | Trocken Hefe |
| 2 El | Olivenöl |
